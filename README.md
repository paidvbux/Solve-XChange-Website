# How to use GitHub

## Using GitHub:

1. Signup for a GitHub account.
2. Download GitHub Desktop.
3. Sign in to your account on GitHub Desktop

## How to fork this repository (How to get the code):

1. Click "Clone a Repository from the Internet...".
5. Select "URL".

7. Click "Clone".
